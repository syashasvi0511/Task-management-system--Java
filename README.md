# 🎯 Student Task Manager (Java)

## 📌 What This Project Does

This project is a simple Student Task Manager built using Java. I created it to help students keep track of their daily tasks, assignments, and deadlines in an organized way.

It allows users to add new tasks, view them, mark them as completed, search for specific tasks, and sort them based on priority. The goal is to make task management easier and more structured.

## ▶️ How to Use the Program

After running the program, you will see a menu like this:

```
STUDENT TASK MANAGEMENT SYSTEM
1. New Task
2. View Tasks
3. Mark Completed
4. Delete Task
5. Search Task
6. Sort by Priority
7. Exit
```

### You can:

* Press **1** → Add a new task (title, description, deadline, priority)
* Press **2** → View all tasks
* Press **3** → Mark a task as completed
* Press **4** → Delete a task
* Press **5** → Search for tasks using keywords
* Press **6** → Sort tasks based on priority
* Press **7** → Exit the program
